# django 연습

[django 가이드 바로가기](https://docs.djangoproject.com/ko/3.2/intro/)

장고 프로젝트 생성
```
django-admin startproject mysite
```

장고 개발 서버 열기
```
py manage.py runserver 8000
```

앱 생성
```
py manage.py startapp {app이름}
```
